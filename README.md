# Users Dashboard

A responsive CRUD dashboard built with React and Tailwind CSS.  
Users can search, sort, paginate, add, edit, and delete records.

## Features

- View users in table format
- Search users by name
- Sort by Name and Email
- Pagination
- Add new user
- Edit existing user
- Delete user with confirmation
- Responsive design
- Premium dark UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## Setup Steps

1. Clone the repository

git clone <your-repo-link>

2. Open project folder

cd your-project-name

3. Install dependencies

npm install

4. Run development server

npm run dev

5. Open browser

http://localhost:5173
